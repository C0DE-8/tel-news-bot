# Talend API Tester collection notes

Live base URL:

```text
https://tel-news-bot-ivory.vercel.app
```

For JSON requests also add:

```text
Content-Type: application/json
```

Admin/test/webhook-management routes use `TELEGRAM_ADMIN_CHAT_IDS` from the server environment. You do not add that value as a Talend header.

On Vercel, also set this environment variable so the bot always knows which group to post into:

```text
TELEGRAM_GROUP_CHAT_IDS=-4586389005|Main News Group
```

Use the real Telegram group id. Group ids are usually negative numbers.

## 1. Health Check

Purpose: Check that the deployed API is online.

```text
Method: GET
URL: https://tel-news-bot-ivory.vercel.app/health
Headers: none
Body: none
```

Expected:

```json
{ "ok": true }
```

## 2. Set Telegram Webhook

Purpose: Tell Telegram to send bot updates to your live Vercel API.

```text
Method: POST
URL: https://tel-news-bot-ivory.vercel.app/webhook/telegram/set
Headers:
  Content-Type: application/json
Body:
```

```json
{
  "url": "https://tel-news-bot-ivory.vercel.app/webhook/telegram",
  "dropPendingUpdates": true
}
```

Use `dropPendingUpdates: true` after a webhook error like `500 Internal Server Error`. That clears Telegram's retry queue so old failed updates do not keep replaying.

Body: none

## 5. List Known Groups

Purpose: See groups the bot has learned from Telegram updates.

```text
Method: GET
URL: https://tel-news-bot-ivory.vercel.app/admin/groups
Headers: none
Body: none
```

Important: Telegram does not let bots list every group they are in. The bot learns a group after it receives `my_chat_member` from Telegram or any message/update from that group.

## 6. Telegram Webhook Receiver

Purpose: This is the route Telegram calls. You normally do not call this manually except for simulation.

```text
Method: POST
URL: https://tel-news-bot-ivory.vercel.app/webhook/telegram
Headers:
  Content-Type: application/json
Body:
```

```json
{
  "update_id": 10030,
  "my_chat_member": {
    "chat": {
      "id": "-4586389005",
      "type": "group",
      "title": "My Test Group"
    },
    "from": {
      "id": "6112214313"
    },
    "old_chat_member": {
      "status": "left"
    },
    "new_chat_member": {
      "status": "administrator"
    }
  }
}
```

Use this only to simulate Telegram telling the bot it was added to a group.

## 9. Send Test Message To Group

Purpose: Verify the bot can send a real message into the selected group.

Important: `chatId` must be the group id, not your private Telegram user/admin id. If you put your private id here, Telegram will send the message to you.

```text
Method: POST
URL: https://tel-news-bot-ivory.vercel.app/admin/group-test-message
Headers:
  Content-Type: application/json
Body:
```

```json
{
  "chatId": "-4586389005",
  "text": "Test message from the news bot."
}
```

## 10. Configure News Posting

Purpose: Configure the group to receive news over time.

Important: `chatId` must be the group id, not your private Telegram user/admin id.

```text
Method: POST
URL: https://tel-news-bot-ivory.vercel.app/admin/news-config
Headers:
  Content-Type: application/json
Body:
```

```json
{
  "chatId": "-4586389005",
  "topic": "crypto",
  "intervalMinutes": 30,
  "postLimit": 10,
  "postNow": false
}
```

Allowed topics:

```text
crypto
politics
```

## 12. Configure And Post Immediately

Purpose: Save config and immediately send one news item.

Manual immediate posts have a 1-minute cooldown per group. If you trigger another immediate post too soon, the bot will ask you to wait.

```text
Method: POST
URL: https://tel-news-bot-ivory.vercel.app/admin/news-config
Headers:
  Content-Type: application/json
Body:
```

```json
{
  "chatId": "-4586389005",
  "topic": "crypto",
  "intervalMinutes": 30,
  "postLimit": 3,
  "postNow": true
}
```

## 16. Delete Telegram Webhook

Purpose: Remove webhook if you want to switch back to polling/local development.

```text
Method: DELETE
URL: https://tel-news-bot-ivory.vercel.app/webhook/telegram
Headers: none
Body: none
```

## Recommended Start Order

1. `GET /health`
2. `POST /webhook/telegram/set`
3. `GET /webhook/telegram/info`
4. Add bot to your Telegram group and make it admin
5. Send any message in the group, or wait for Telegram `my_chat_member`
6. `GET /admin/groups`
7. `POST /admin/group-check`
8. `POST /admin/group-test-message`
9. `POST /admin/news-config`
